The Franklin Community Co-op PriceCheck web site is enabled on this device.

The PriceCheck ASP server handles basic price checking functionality.
Product barcode lookups require a single HTTP "Post" parameter of
?nnnnnnnnnn or ?code=nnnnnnnnnn where nnn... represents a barcode
that exists in the server database.

A successful lookup will return a web page that contains the product description
and item price.

A failed lookup will simply return an "Item not found" web page.
